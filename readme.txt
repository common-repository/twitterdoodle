== Description ==

Allows a user to create posts and categories based on Twitter keyword searches.  Each post would be a digest of twitter posts in that category. 


== Installation ==

1. Upload the `twitterdoodle` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Configuration ==

1. Go to WordPress settings, and select the Twitterdoodle category. 
1. Set the general Options that will be applied to all TwitterDoodle posts.
1. Set up up your auto posts based on keyword, keywords, or keyword phrase. 
1. Pick the category for the TwitterDoodle posts. 
1. Select the interval or timing of when keyword posts go up automatically. 
1. Select whether or not you want the TwitterDoodle post to appear on the front page of your blog or only in the category.
